 Primero el codigo importa la libreria re, que es de expresiones regulares.
 Se define la funcion palindrome que tiene de argumento la lista palabras. Se inicializa un contador llamado resultado.Se implementa la utilizacion de un bucle for que recorre cada palabra de la lista palabras, para cada palabra se aplica la libreria Re que lo que hace es una limpieza para eliminar todos los caracteres no alfanumericos y convertir en minuscula, es una funcion re.sub de la libreria la que la realiza, despues se compara la palabra original con su version invertida, que se compararia la palabra limpia y si es igual entonces es palindrome y se suma 1 a resultado que cuenta la cantidad de palabras palindromes.
